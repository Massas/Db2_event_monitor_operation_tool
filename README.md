# Db2_event_monitor_operation_tool

Db2のCUI監視ツールであるイベントモニターの管理・操作を行うシェルスクリプトのリポジトリです。

機能は大きく以下の8つです
１．イベントモニターの作成
２．イベントモニターの削除
３．イベントモニターの起動
４．イベントモニターの停止
５．イベントモニターの設定・状況確認
６．イベントモニターのフォーマット
７．イベントモニターのアウトプット削除
８．スクリプト終了

対応予定シェル：
ksh(文字化け1件残で動作には影響なし)
bash(実装中)

DBNAME、ID、PWDについては各自で適当な値に書き換えて使用してください。
